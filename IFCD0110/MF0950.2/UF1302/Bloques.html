    • HTML para citas cortas

    El HTML <q> elemento define un presupuesto corto.

    Los navegadores suelen insertar comillas alrededor del <q> elemento.

    Ejemplo

    WWF's goal is to: Build a future where people live in harmony with nature.

    HTML <blockquote> para citas largas

    El HTML <blockquote> elemento define una sección citado.

    Navegadores generalmente guión <blockquote> elementos.

    Ejemplo

    <Here is a quote from WWF's website:> <blockquote cite="http://www.worldwildlife.org/who/index.html"> For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally. </blockquote>

    HTML <abbr> de abreviaturas

    El HTML <abbr> elemento define una abreviatura o un acrónimo.

    abreviaturas de marcado pueden proporcionar información útil a los navegadores, sistemas de traducción y los motores de búsqueda.

    Ejemplo

    <p>The <abbr title="World Health Organization"> WHO </abbr> was founded in 1948.</p>

    HTML <address> para información de contacto

    El HTML <address> elemento define la información de contacto (author/owner) de un documento o artículo.

    La <address> elemento es usualmente representada en cursiva. La mayoría de los navegadores añadir un salto de línea antes y después del elemento.

    Ejemplo

    <address> Written by John Doe.
    Visit us at:
    Example.com
    Box 564, Disneyland
    USA

    HTML para la anulación bidireccional

    El HTML elemento define override bi-direccional.

    La elemento se utiliza para anular la dirección del texto actual:

    The <div> Element

    The <div> element is often used as a container for other HTML elements.

    The <div> element has no required attributes, but style, class and id are common.

    When used together with CSS, the <div> element can be used to style blocks of content:

    Example

    <div style="background-color:black;color:white;padding:20px;"> <h2>London</h2> <p>London is the capital city of England. It is the most populous city in the United Kingdom, with a metropolitan area of over 13 million inhabitants.</p> </div>

    The <span> Element

    The <span> element is often used as a container for some text.

    The <span> element has no required attributes, but style, class and id are common.

    When used together with CSS, the <span> element can be used to style parts of the text:

    Example

    <h1>My <span style="color:red">Important</span> Heading</h1>
